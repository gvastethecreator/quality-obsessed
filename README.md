# Quality Obsessed

![Quality Obsessed banner](./assets/readme-banner.png)

> Codex skill pack for evidence-gated quality work: inspect the real artifact, improve it directly, and prove the result before calling it done.

Use it when the task calls for obsessive, evidence-backed quality instead of a competent first pass.

## Install

Copy [`SKILLS/quality-obsessed`](./SKILLS/quality-obsessed) into your Codex skills directory, or ask Codex to install `quality-obsessed`.

## Core Contract

- Inspect the real artifact: runtime, screenshots, output, tests, states, or edge cases.
- Change the artifact, not just the plan.
- Prove each quality claim with evidence.
- Beat the baseline, or say `failed` / `red/failed`.
- If meaningful weaknesses remain and the user gave no limit: 30-loop floor, no hard cap, Loop 30 verdict (`continue`, `ask`, `stop`).
- Use council, grilling, delegated review, and visual horizons when they improve the artifact, decision, proof, or cuts.
