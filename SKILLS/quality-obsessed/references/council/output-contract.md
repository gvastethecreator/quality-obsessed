# Output Contract

Use these shapes for council outputs. Keep them concise unless the user asked for a full PRD.

## Question Round

Ask only questions that change scope, direction, risk, or user satisfaction.

```text
Decision needed:
Recommended default:
Why this matters:
Alternatives:
Impact if unanswered:
Settled context:
Execution blocked until:
```
