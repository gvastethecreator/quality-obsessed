# Skills

This repository ships one portable Agent Skills package:

- [quality-obsessed](./quality-obsessed/SKILL.md): evidence-gated improvement with safe mission modes, canonical task/artifact/verification states, applicable gates, explicit Creative Search for standout/greenfield direction work, dynamic 10-loop deep persistence, role-based model routing, recovery, domain profiles, and host-capability mapping for Codex, Claude Code, OpenCode, and compatible agents.

The standalone repository is the canonical source. Install with the Skills CLI or copy the package into the skill directory supported by your host.
